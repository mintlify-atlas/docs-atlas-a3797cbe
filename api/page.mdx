title: Page
description: Page class API reference for creating custom pages in Filament panels
---

The `Page` class is the foundation for creating custom pages in Filament panels. Pages can include forms, widgets, actions, and custom content, and can be organized within navigation groups and clusters.

## Namespace

```php
Filament\Pages\Page
```

## Inheritance

Extends: `Filament\Pages\BasePage`

Implements: `HasActions`, `HasRenderHookScopes`, `HasSchemas`

## Type Parameters

- `TConfiguration` - The page configuration class (extends `PageConfiguration`)

## Static Methods

### getUrl()

Generate a URL to this page.

```php
public static function getUrl(
    array $parameters = [],
    bool $isAbsolute = true,
    ?string $panel = null,
    ?Model $tenant = null,
    bool $shouldGuessMissingParameters = false,
    ?string $configuration = null
): string
```

<ParamField path="parameters" type="array<mixed>" default="[]">
  Additional route parameters.
</ParamField>

<ParamField path="isAbsolute" type="bool" default="true">
  Whether to generate an absolute URL.
</ParamField>

<ParamField path="panel" type="string | null" default="null">
  The panel ID. Defaults to the current panel.
</ParamField>

<ParamField path="tenant" type="Model | null" default="null">
  The tenant model for multi-tenancy.
</ParamField>

<ParamField path="shouldGuessMissingParameters" type="bool" default="false">
  Whether to guess missing route parameters.
</ParamField>

<ParamField path="configuration" type="string | null" default="null">
  The configuration key.
</ParamField>

<ParamField path="return" type="string">
  Returns the generated URL.
</ParamField>

### content()

Define the content schema for this page.

```php
public function content(Schema $schema): Schema
```

<ParamField path="schema" type="Schema">
  The schema instance to configure.
</ParamField>

<ParamField path="return" type="Schema">
  Returns the configured schema.
</ParamField>

**Example:**

```php
use Filament\Schemas\Schema;
use Filament\Schemas\Components\Section;

public function content(Schema $schema): Schema
{
    return $schema
        ->components([
            Section::make('Content')
                ->schema([
                    // Components...
                ]),
        ]);
}
```
