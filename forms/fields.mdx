## Introduction

Filament provides a comprehensive set of form fields for capturing different types of data. All form field classes are found in the `Filament\Forms\Components` namespace.

## Text Input

The text input allows you to interact with string values:

```php
use Filament\Forms\Components\TextInput;

TextInput::make('name')
    ->required()
    ->maxLength(255)
```

### Input types

You can set the HTML input type using dedicated methods:

<CodeGroup>

```php Email
TextInput::make('email')
    ->email()
    ->required()
```

```php URL
TextInput::make('website')
    ->url()
    ->prefix('https://')
```

```php Telephone
TextInput::make('phone')
    ->tel()
    ->telRegex('/^[+]*[(]{0,1}[0-9]{1,4}[)]{0,1}[-\s\./0-9]*$/')
```

```php Password
TextInput::make('password')
    ->password()
    ->revealable()
    ->required()
```

```php Numeric
TextInput::make('price')
    ->numeric()
    ->step(0.01)
    ->prefix('$')
```

```php Integer
TextInput::make('quantity')
    ->integer()
    ->minValue(1)
    ->maxValue(100)
```

</CodeGroup>

### Affixes

You can add prefixes and suffixes to text inputs:

```php
TextInput::make('domain')
    ->prefix('https://')
    ->suffix('.com')

TextInput::make('price')
    ->numeric()
    ->prefix('$')
    ->suffix('USD')
```

### Input masks

Apply input masks for formatted input:

```php
TextInput::make('phone')
    ->mask('(999) 999-9999')

TextInput::make('credit_card')
    ->mask('9999 9999 9999 9999')
```

## Select

The select component allows selection from predefined options:

```php
use Filament\Forms\Components\Select;

Select::make('status')
    ->options([
        'draft' => 'Draft',
        'reviewing' => 'Reviewing',
        'published' => 'Published',
    ])
    ->required()
```

### Searchable selects

Enable search for easier option access:

```php
Select::make('author_id')
    ->label('Author')
    ->options(User::pluck('name', 'id'))
    ->searchable()
```

### Multi-select

Allow multiple selections:

```php
Select::make('categories')
    ->multiple()
    ->options(Category::pluck('name', 'id'))
    ->searchable()
```

### Native vs JavaScript select

By default, Filament uses native HTML selects. Enable a JavaScript-powered select:

```php
Select::make('status')
    ->options([...])
    ->native(false)
```

### Relationship selects

Populate options from Eloquent relationships:

```php
Select::make('author_id')
    ->relationship('author', 'name')
    ->searchable()
    ->preload()
```

## Checkbox

A single checkbox for boolean values:

```php
use Filament\Forms\Components\Checkbox;

Checkbox::make('is_active')
    ->label('Active')
    ->inline(false)
```

## Toggle

A modern toggle switch for boolean values:

```php
use Filament\Forms\Components\Toggle;

Toggle::make('is_featured')
    ->label('Featured')
    ->onIcon('heroicon-m-star')
    ->offIcon('heroicon-m-star')
    ->onColor('warning')
```

## Radio

Radio buttons for single selection from options:

```php
use Filament\Forms\Components\Radio;

Radio::make('status')
    ->options([
        'draft' => 'Draft',
        'scheduled' => 'Scheduled',
        'published' => 'Published',
    ])
    ->descriptions([
        'draft' => 'Not visible to the public.',
        'scheduled' => 'Will be published at the scheduled time.',
        'published' => 'Visible to the public.',
    ])
```

## Checkbox List

Multiple checkboxes for array selection:

        'tailwind' => 'Tailwind CSS',
        'alpine' => 'Alpine.js',

## Date & Time Pickers

Date, time, and datetime selection:

<CodeGroup>

```php Date Picker
use Filament\Forms\Components\DatePicker;

DatePicker::make('date_of_birth')
    ->native(false)
    ->displayFormat('d/m/Y')
    ->minDate(now()->subYears(100))
    ->maxDate(now())
```

```php Time Picker
use Filament\Forms\Components\TimePicker;

TimePicker::make('start_time')
    ->native(false)
    ->seconds(false)
```

```php DateTime Picker
use Filament\Forms\Components\DateTimePicker;

DateTimePicker::make('published_at')
    ->native(false)
    ->timezone('America/New_York')
    ->minDate(now())
```

</CodeGroup>

## File Upload

File and image upload with preview:

```php
use Filament\Forms\Components\FileUpload;

FileUpload::make('avatar')
    ->image()
    ->imageEditor()
    ->imageEditorAspectRatios([
        '16:9',
        '4:3',
        '1:1',
    ])
    ->maxSize(1024)
```

### Multiple file uploads

```php
FileUpload::make('attachments')
    ->multiple()
    ->maxFiles(5)
    ->acceptedFileTypes(['application/pdf', 'image/*'])
```

### Image uploads

```php
FileUpload::make('cover_image')
    ->image()
    ->imagePreviewHeight('250')
    ->imageResizeMode('cover')
    ->imageCropAspectRatio('16:9')
    ->imageResizeTargetWidth('1920')
    ->imageResizeTargetHeight('1080')
```

## Repeater

Repeatable set of fields for array data:

```php
use Filament\Forms\Components\Repeater;
use Filament\Forms\Components\TextInput;
use Filament\Forms\Components\Select;

Repeater::make('team_members')
    ->schema([
        TextInput::make('name')
            ->required(),
        Select::make('role')
            ->options([
                'developer' => 'Developer',
                'designer' => 'Designer',
                'manager' => 'Manager',
            ])
            ->required(),
    ])
    ->columns(2)
    ->defaultItems(1)
    ->addActionLabel('Add team member')
    ->collapsible()
```

### Repeater with relationships

```php
Repeater::make('qualifications')
    ->relationship()
    ->schema([
        TextInput::make('title')->required(),
        DatePicker::make('issued_at')->required(),
    ])
```

## Builder

Flexible builder with multiple block types:

```php
use Filament\Forms\Components\Builder;
use Filament\Forms\Components\Builder\Block;
use Filament\Forms\Components\TextInput;
use Filament\Forms\Components\Textarea;
use Filament\Forms\Components\FileUpload;

Builder::make('content')
    ->blocks([
        Block::make('heading')
            ->schema([
                TextInput::make('content')
                    ->label('Heading')
                    ->required(),
                Select::make('level')
                    ->options([
                        'h1' => 'Heading 1',
                        'h2' => 'Heading 2',
                        'h3' => 'Heading 3',
                    ])
                    ->required(),
            ]),
        Block::make('paragraph')
            ->schema([
                Textarea::make('content')
                    ->label('Paragraph')
                    ->required(),
            ]),
        Block::make('image')
            ->schema([
                FileUpload::make('url')
                    ->label('Image')
                    ->image()
                    ->required(),
                TextInput::make('alt')
                    ->label('Alt text')
                    ->required(),
            ]),
    ])
    ->collapsible()
```

## View Field

Render a custom Blade view as a field:

```php
use Filament\Forms\Components\ViewField;

ViewField::make('custom_content')
    ->view('forms.components.custom-field')
```

## Placeholder

Display read-only content:

```php
use Filament\Forms\Components\Placeholder;

Placeholder::make('created_at')
    ->content(fn ($record): string => $record?->created_at?->diffForHumans() ?? '-')
```
